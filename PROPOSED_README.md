## üåü Project Overview

MementoMind is an innovative browser extension that transforms your web browsing by adding contextual notes, tags, and Wayback Machine snapshots to your bookmarks. Enhance recall, organize pages, and rediscover the 'why' behind every saved link with powerful organization and privacy features.

---

## ‚ú® Key Features

*   **Contextual Bookmarking:** Attach notes and tags directly to saved URLs.
*   **Wayback Machine Integration:** Capture and link to historical snapshots of web pages.
*   **Enhanced Recall:** Quickly remember the context and purpose of saved links.
*   **Privacy-Focused:** Your data is stored locally or synced securely.
*   **Cross-Browser Compatibility:** Works on Chrome and Firefox.
*   **Manifest V3 Compliant:** Built with modern browser extension standards.

## üöÄ Getting Started

### Prerequisites

*   Node.js (v20+ recommended)
*   npm or Yarn (comes with Node.js)
*   Your preferred browser (Chrome, Firefox, Edge)

### Installation

1.  **Clone the repository:**
    bash
    git clone https://github.com/chirag127/MementoMind-Contextual-Web-Memory-Browser-Extension.git
    cd MementoMind-Contextual-Web-Memory-Browser-Extension
    

---

## üõ†Ô∏è Development Setup

1.  **Install Dependencies:**
    bash
    npm install
    # or
    yarn install
    

2.  **Run in Development Mode:**
    This command will build the extension and provide a watch mode for automatic reloads.
    bash
    npm run dev
    # or
    yarn dev
    

3.  **Load the Extension in your Browser:**
    *   **Chrome/Edge:** Navigate to `chrome://extensions/` or `edge://extensions/`, enable "Developer mode", and click "Load unpacked". Select the `dist` or `build` folder from the project root.
    *   **Firefox:** Navigate to `about:debugging#/runtime/this-firefox`, click "Load Temporary Add-on...", and select any file from the project's `dist` or `build` directory.

---

## ‚ñ∂Ô∏è Usage

Once installed, MementoMind provides a seamless experience:

1.  **Save a Page:** Click the MementoMind icon in your browser toolbar to save the current page.
2.  **Add Context:** In the popup, add notes and tags to your saved bookmark.
3.  **View Snapshots:** Access Wayback Machine snapshots directly from your bookmark list.
4.  **Organize:** Filter and search your saved memories by tags or keywords.
